La fonctionnalité de suivi de piste peut être codé dans le fichier "RXmodule\RXviewerPathTracking.py".
Il y a des explications de certaines variables utile.

La fonction "pathTracking" dans ce fichier a été reliée à la boîte à outils avec l'icône de la baguette magique.

Pour exécuter le programme python lancer main.py.



Pour transformer les fichiers python en exécutable puis en setup (il faut que le dossier 'RXViewer' soit sur le bureau):
	- vidé la contenue du fichier "asset\recent"
	- exécuter la commande "py to exe\cmd.bat"
	- ouvrir "py to exe\setup.iss" avec le logiciel "Inno Setup Compiler" et run le script
	- vous avez maintenant votre fichier setup sur le bureau
